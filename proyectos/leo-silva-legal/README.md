# Leo Silva Legal

**Sitio web corporativo para estudio jurídico**  
**Tipo:** Proyecto web / presencia digital profesional  
**Estado:** En desarrollo / mejora continua

---

## Descripción

Leo Silva Legal es un sitio web corporativo orientado a la presentación de servicios legales, fortalecimiento de presencia digital y captación de clientes para un estudio jurídico.

El proyecto busca ofrecer una imagen profesional, clara y confiable, mostrando información institucional, áreas de servicio, canales de contacto y estructura web adaptada a las necesidades de un despacho legal.
