# Cupotech Solutions

**Proyecto de desarrollo web, posicionamiento digital e integración de sistema interno**  
**Año:** 2024 - Actualidad  
**Tipo:** Proyecto web / sistema interno / documentación funcional

---

## Descripción

Cupotech Solutions es un proyecto tecnológico orientado a soporte técnico, mantenimiento, desarrollo web y soluciones de software.

El proyecto inició con una primera versión web en WordPress, enfocada en presencia digital, posicionamiento SEO y medición del dominio mediante herramientas de Google. Posteriormente, se desarrolló y desplegó una nueva versión web informativa en HTML5, CSS3 y JavaScript, actualmente publicada en hosting y orientada a la presentación de servicios, membresías, contacto comercial y redirección a WhatsApp.

Actualmente, el proyecto contempla la integración de un sistema interno desarrollado en Java con Spring Boot y MySQL, que permitirá ampliar la web hacia funcionalidades dinámicas como panel de cliente, intranet administrativa, gestión de membresías, carrito de compra, servicios técnicos, tickets, usuarios, roles, historial de atención y reportes.
