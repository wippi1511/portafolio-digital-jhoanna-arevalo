# Proyecto SMEAL

**Solución web para gestión de compras, ventas, almacén y tienda online**  
**Tipo:** Proyecto académico / concurso institucional  
**Reconocimiento:** 2do lugar a nivel institucional IDAT

---

## Descripción

SMEAL es una solución web integral orientada a la gestión de compras, ventas, almacén y tienda online.

El sistema permite administrar productos, controlar operaciones comerciales, gestionar el flujo de compra en línea y conectar el carrito de compras con una pasarela de pagos, integrando funcionalidades de backend, base de datos y frontend.

---

## Mi participación

- Participé en el desarrollo backend del sistema utilizando C#/.NET.
- Desarrollé funcionalidades relacionadas con productos, pedidos, carrito de compras y flujo de compra.
- Participé en la integración del carrito de compras con la pasarela de pagos.
- Diseñé y construí la base de datos del sistema.
- Implementé tablas, relaciones y procedimientos almacenados para compras, ventas, almacén, clientes y operaciones transaccionales.
- Coordiné la integración entre backend, base de datos y frontend para asegurar el funcionamiento de la tienda online.
