# TRENTO ERP / Techera

**Migración de sistema monolítico a servicios .NET**  
**Tipo:** Proyecto profesional / backend / base de datos  
**Periodo:** 2026

---

## Descripción

TRENTO ERP es un proyecto orientado a la migración de funcionalidades de un sistema ERP monolítico hacia una arquitectura por capas, orientada a servicios y microservicios en .NET/C#.

El trabajo se enfocó en la modernización de componentes backend, integración con base de datos, optimización de consultas, uso de procedimientos almacenados y mejora de la estructura técnica del sistema.

> Por confidencialidad del proyecto, este repositorio presenta únicamente documentación resumida. No se publica código fuente ni información interna de la empresa.

---

## Mi participación

- Participé como practicante de desarrollo backend C#.
- Desarrollé y ajusté componentes backend utilizando C#, .NET, Dapper, SQL Server y procedimientos almacenados.
- Apoyé en la implementación y mantenimiento de controladores, servicios, repositorios, DTOs, entidades y mapeos con AutoMapper.
- Realicé integración de consultas y operaciones con base de datos mediante stored procedures.
- Apoyé en la revisión y corrección de errores relacionados con endpoints, tiempos de respuesta, validaciones, serialización de datos y conexión con base de datos.
- Utilicé Git, GitHub, GitLab y consola para control de versiones, gestión de cambios y seguimiento técnico.

## Funcionalidades trabajadas

- Migración de funcionalidades desde estructura monolítica.
