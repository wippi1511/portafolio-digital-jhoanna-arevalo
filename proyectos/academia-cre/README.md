# Academia CRE

**Plataforma web académica para gestión de cursos e inscripciones**  
**Tipo:** Proyecto web / sistema académico  
**Estado:** En desarrollo / mejora continua

---

## Descripción

Academia CRE es una plataforma web orientada a la gestión de cursos, usuarios, inscripciones y paneles por rol. El sistema fue desarrollado para organizar la oferta educativa, facilitar el proceso de matrícula de alumnos y mejorar la administración interna de cursos, profesores, alumnos e inscripciones.

El proyecto contempla una estructura web con funcionalidades de registro, inicio de sesión, gestión de cursos, control de usuarios, estados de matrícula, materiales académicos y flujo comercial para inscripción mediante WhatsApp y medios de pago como Yape o transferencia.

## Funcionalidades principales

- Registro e inicio de sesión de usuarios.
- Roles de alumno, profesor y administrador.
- Gestión de cursos y materiales académicos.
- Inscripción de alumnos a cursos.
- Control de estados de matrícula y pagos.
- Paneles diferenciados por rol.
- Integración de flujo comercial con WhatsApp.
- Referencia de medios de pago como Yape o transferencia.
- Administración de contenido académico.
