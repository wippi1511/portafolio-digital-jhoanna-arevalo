# Majo Fotografía

**Sitio web para portafolio visual y servicios fotográficos**  
**Tipo:** Proyecto web / presencia digital  
**Estado:** En desarrollo / mejora continua

---

## Descripción

Majo Fotografía es un sitio web orientado a la presentación de servicios fotográficos, portafolio visual, información de contacto y presencia digital.

El proyecto busca mostrar de forma atractiva los trabajos fotográficos, facilitar la comunicación con potenciales clientes y fortalecer la imagen profesional de la marca mediante una web informativa, visual y responsive.

---
