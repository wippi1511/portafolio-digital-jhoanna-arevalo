# Sistema de Gestión Anibella

**Aplicación de escritorio para gestión comercial e inventario**  
**Tipo:** Proyecto independiente  
**Estado:** En mantenimiento

---

## Descripción

Sistema de escritorio desarrollado para la gestión operativa de una tienda de ferretería, orientado al control de inventario, ventas, clientes, créditos, deudas, abonos y caja.

El sistema permite centralizar la información comercial del negocio mediante una aplicación de escritorio conectada a una base de datos MySQL alojada en hosting.

---

## Mi participación

- Desarrollé la aplicación de escritorio en C#/.NET.
- Diseñé e integré la base de datos en MySQL.
- Implementé módulos para inventario, ventas, clientes, créditos, caja y reportes.
- Configuré la conexión del sistema con base de datos alojada en hosting.
- Realizo mantenimiento continuo del sistema y de la base de datos.

---

## Funcionalidades principales

- Registro, actualización, eliminación y consulta de productos.
- Control de inventario y existencias.
- Registro de ventas.
- Gestión de clientes.
- Control de créditos, deudas y abonos.
- Control de caja diaria, quincenal, mensual y semestral.
- Reportes operativos y financieros.
- Mantenimiento de base de datos.
